15 ## Installation
16 
17  mkdir build
18  cd build
19  cmake .. -DCMAKE_INSTAL_PREFIX="path/to/install/location"
20  make install
21 
22 Use 'softinstall' to create symlinks instead of copies.
23 
24 If this path is not yet in the CMake module path, add to CMakeLists.txt of your project:
25 
26  list(APPEND CMAKE_MODULE_PATH /path/to/cmakescripts)
27 
28 And then include the desired script (e.g. softinstall) with:
29 
30  include(fsc/softinstall)
