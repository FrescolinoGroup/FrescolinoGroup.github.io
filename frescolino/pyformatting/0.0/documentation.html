fsc.formatting.sstr(obj, length=50)[source]¶

Returns the str representation of an object, and shortens it, if it is too long.

Args:
obj: Any python object that we want to convert to a string. length (int): The maximal length of the resulting string.
Returns:
str. This string is guaranteed to have maximal size length.
