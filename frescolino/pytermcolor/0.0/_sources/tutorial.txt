Tutorial
========

To import the package you can use:

.. code:: python

    from fsc import termcolor

If you want to print "Hello World" colored red and blue, use the color functions.

.. code:: python

    print(termcolor.red("Hello"), termcolor.blue("World"))
    
The color functions take any argument which is convertible to string.

.. code:: python

    print(termcolor.green(1), termcolor.cyan([1,2,3]))
    
For precise control one can use the color dictionary.

.. code:: python

    print("{red}{} {blue}{} {green}{:.2}{none}".format("Hello", "World", 3.1453, **termcolor.color))

The none entry resets the color, otherwise the last active color will be used in further print statements.
