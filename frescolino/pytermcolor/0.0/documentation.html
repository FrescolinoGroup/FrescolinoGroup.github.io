Documentation¶

fsc.termcolor.enable()[source]¶

Enables the colors by adding the correct ANSI escape sequence to the color dictionary.

fsc.termcolor.disable()[source]¶

Disables the colors by adding the empty strings to the color dictionary.

fsc.termcolor.none(obj)¶

Resets the color settings to normal.

fsc.termcolor.clrscr(obj)¶

Clears the screen.

fsc.termcolor.clear(obj)¶

Jumps to the beginnig of the line, i.e. you can overwrite text.

fsc.termcolor.renter(obj)¶

Undoes a newline, i.e. you can overwrite an already printed line.
